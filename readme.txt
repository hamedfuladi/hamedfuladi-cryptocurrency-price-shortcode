Instantly add live cryptocurrency prices to your WordPress site with Cryptocurrency Price Shortcode.

== Description ==
The Cryptocurrency Price Shortcode plugin allows you to easily embed live cryptocurrency prices into your WordPress site using shortcodes. With this plugin, users can select their desired cryptocurrency and destination currency from a list of the top 200 cryptocurrencies and 58 destination currencies, respectively. The plugin generates a shortcode based on the user\'s selection, allowing them to effortlessly display live cryptocurrency prices on any page or post.


== Features ==

- **Generate shortcode to display live cryptocurrency prices.
- **Select from a list of the top 200 cryptocurrencies and 58 destination currencies.
- **Copy shortcode to clipboard with the click of a button.
- **Display live cryptocurrency prices in the selected destination currency on any page or post.

== Installation ==
1. Download the plugin zip file.
2. Upload the plugin zip file to the WordPress admin dashboard.
3. Activate the plugin through the Plugins menu in WordPress.
